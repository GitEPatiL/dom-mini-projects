# DOM Mini Projects

Welcome to the **DOM Mini Projects** repository! This collection features 5 beginner-friendly web development projects designed to help you practice and master Document Object Model (DOM) manipulation using vanilla HTML, CSS, and JavaScript.

## 🚀 Projects Included

1. **Color Switcher** (`/color switcher`)
   A simple web application that allows users to change the background color of the page using multiple buttons. A great introduction to event listeners and modifying CSS properties via JavaScript.

2. **Animated Cursor** (`/Animated Cursor`)
   A project demonstrating how to override the default pointer and create a custom, interactive animated mouse cursor using JavaScript mouse tracking and CSS animations.

3. **List and Search** (`/List and Search`)
   An interactive application that displays a list of items and allows the user to dynamically filter and search through them in real-time as they type.

4. **Simple Calculator** (`/Simple Calculator`)
   A fully functional basic calculator with an intuitive interface. Practice handling user inputs, performing arithmetic operations, and updating the display using pure JavaScript.

5. **Simple Todo List** (`/Simple TOdo List`)
   A classic task management application. Users can add new tasks, mark them as complete, and delete them. Excellent practice for creating, reading, updating, and deleting DOM elements dynamically.

## 🛠️ Technologies Used

- **HTML5**: For crafting the structural layout of the web pages.
- **CSS3**: For styling the components, managing layouts, and adding visual flair.
- **Vanilla JavaScript**: For implementing the interactive logic, event handling, and DOM manipulation (no external libraries or frameworks).

## 🏃‍♂️ How to Run

Since these projects are built with core web technologies, there's no complex build process, server setup, or installation required.

1. Clone or download this repository to your local machine.
2. Navigate into the specific project folder you wish to explore (e.g., `color switcher`).
3. Double-click the `index.html` file to open it directly in any modern web browser.

## 🎯 Learning Outcomes

Exploring and extending these projects will help you practice:
- Selecting DOM elements (`querySelector`, `getElementById`, etc.).
- Attaching and handling user interactions (`addEventListener`).
- Dynamically modifying styles and CSS classes (`classList.add/remove`, `style.backgroundColor`).
- Creating and removing HTML elements using JavaScript (`createElement`, `appendChild`, `remove`).
